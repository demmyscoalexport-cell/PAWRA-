## 🔧 CUSTOMIZATION IN 5 MINUTES

### Change Brand Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  ivory: '#FFFBF5',  // Change these
  'soft-emerald': '#4A6B5C',
  // Add your colors
}
```

### Connect Your Shopify Store
Edit `.env`:
```
VITE_SHOPIFY_STORE_DOMAIN=your-store.myshopify.com
VITE_SHOPIFY_STOREFRONT_TOKEN=your_token
```

### Add Your Products
Replace mock data in `src/data/mockProducts.js` or connect to Shopify API.

### Add Your Logo
Replace in `src/components/layout/Header.jsx`

## 💡 EXAMPLE USAGE

### Fetching Products
```jsx
import { useProducts } from '@/hooks/useShopify';

export function ProductList() {
  const { data, isLoading } = useProducts({ first: 20 });
  return <ProductGrid products={data} loading={isLoading} />;
}
```

### Managing Cart
```jsx
import { useCartStore } from '@/stores/cartStore';

const addItem = useCartStore((state) => state.addItem);
const cart = useCartStore((state) => state.cart);

addItem({ id: '1', quantity: 1, merchandise: {...} });
```

### UI State
```jsx
import { useUIStore } from '@/stores/uiStore';

const isCartOpen = useUIStore((state) => state.isCartOpen);
const openCart = useUIStore((state) => state.openCart);

return <button onClick={openCart}>Open Cart</button>;
```
